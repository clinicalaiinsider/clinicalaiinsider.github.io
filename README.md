# Clinical AI Insider

Bridging clinical domain expertise and AI through deep dives into healthcare data ecosystems, NLP pipelines, and real-world evidence generation.

## Focus Areas

### Healthcare Data Interoperability
- **OHDSI/OMOP CDM** - Standardized observational health data
- **HL7 FHIR** - Modern healthcare data exchange
- **Clinical Terminologies** - SNOMED CT, LOINC, RxNorm, ICD-10

### Clinical NLP & AI
- **Named Entity Recognition** - Extracting clinical concepts from text
- **BioBERT & Clinical Transformers** - Domain-specific language models
- **Information Extraction** - Structuring unstructured clinical notes

### Real-World Evidence
- **Cohort Studies** - Patient population analysis with ATLAS
- **Comparative Effectiveness** - Treatment outcome research
- **Data Quality** - Validation and characterization frameworks

## Projects

| Project | Description |

## Tech Stack

- **Data**: PostgreSQL, OMOP CDM, FHIR R4
- **ML/NLP**: Python, Hugging Face, spaCy, BioBERT
- **OHDSI Tools**: ATLAS, Achilles, WebAPI, Broadsea
- **Infrastructure**: Docker, GitHub Actions
